# API Runbook

## Demo sequence

1. Register SME, buyer and financier users.
2. SME creates business profile.
3. Admin verifies business KYB.
4. SME creates order.
5. Buyer confirms order.
6. SME creates invoice.
7. Buyer confirms invoice.
8. SME submits delivery proof with OTP, GPS and buyer confirmation metadata.
9. SME creates receivable.
10. Buyer/financier creates Smart LC.
11. SME/financier calculates credit score and readiness report.
12. Admin reviews proof ledger and blockchain outbox.
